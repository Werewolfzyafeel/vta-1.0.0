VALORANT TACTICAL ASSISTANT [v1.0.0]
Lightweight HUD overlay and input optimization utility for Valorant.

⚡ Key Features
Vector Box HUD (2D/3D): Framing overlay around target models.

Status Bars: Real-time HP and shield tracking on the screen.

Vector Snaplines: Guiding lines from the bottom of the screen to targets.

Smooth Input System: Mouse acceleration and camera panning stabilization to eliminate cursor jitter.

Smart Click-Assistant: Input delay reduction when an entity intersects with your crosshair grid.

Recoil Stabilizer (RSS): Compensates for mouse shake during sustained fire.

🛠 Status & Current Bugs
First public release. Optimized codebase with zero FPS drops.

UI theme is locked to dark-purple (color customization coming in v1.0.1).

Interface is English-only.

Rare overlay flicker when using Alt+Tab (fix by pressing Insert key).

Stabilization profile is global (per-weapon configurations coming soon).
